# 中控杯规划

## 大致步骤

- 策略设计 （抓不抓环 用不用子母机 用不用视觉 用不用机械臂……）
- 机械设计 （根据策略进行设计 以及考虑用什么型材（是否真的要对抗））
- 电路设计 （单片机的选择 几个单片机？ 是否要留通讯接口 传感器的选择（寻线 里程计 视觉 ））
- 第一次测试：运动测试 走直线 转弯 
- 算法设计 （执行策略）
- 跑比赛场地
